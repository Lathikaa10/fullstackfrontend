About RecycleX

RecycleX is a Smart Waste Management and Recycling Platform that helps users recycle waste efficiently. Our system manages recyclable items, collection requests, recycling facilities and processing batches to create a cleaner and greener environment.

Our Mission

To encourage responsible waste management and promote sustainable recycling practices.

Our Vision

To build a smart and eco-friendly society through technology and recycling.

Our Project Modules

Module Description
System User Manage Username, Email, Password and User Role.
Recyclable Item Manage Item Name, Category, Weight, Price and Status.
Collection Request Manage Pickup Requests, Pending Waste and Collection Status.
Recycling Facility Manage Facility Name, Location and Capacity.
Processing Batch Track Batch ID, Processing Date and Batch Status.
